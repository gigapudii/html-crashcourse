Unlike what is commonly mistaken, HTML is not a programming language but a markup language used to instruct browers on how to display specific content by giving it structure and meaning. HTML stands for HyperText Markup Language and is one of the most common languages for the WWW.

Result 1

Posted by Patricia on September 16

Firstly, it has been interesting to see how much the new version still resembles the old in regards to structure and tags. So, what did I finally learn? I began by writing the cheat sheet to get comfortable with HTML5. This included understanding how to save folders on the computer, and create and name the HTML files, i.e. simple and sweet. This gave me an overview of the needed parts of an HTML page (in very general terms: html tag, head, body, footer). A-tags, headings, paragraphs along with some basic decoration such as making a text bold or highlighted was a nice refresh from the old days. It was also beneficial to learn that HTML code also requires indentation, just as any programming language, for the sake of readability.
